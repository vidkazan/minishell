# MINISHELL

Имплементация BASH.

Что реализованно:
-
- команды и аргументы
- редиректы
- сигналы ошибок
- прерывание ввода с помощью CTRL+D, CTRL+C, CTRL+Z
- одинарные и двойные кавычки
- обработка переменных окружения
- мультипайпы
- builtins:
  - echo
  - env
  - export
  - pwd
  - cd

# Запуск
```
  make
  ./minishell
```

для выхода используется команда ``` exit ```
